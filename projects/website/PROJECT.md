# Company Website Project

## Scope
- Landing page with value proposition
- Blog with markdown posts
- Documentation section
- Contact form (linked to Paperclip tickets or email)
- Dark/light mode toggle
- Mobile-responsive

## Tech Stack
- Framework: Astro or Next.js (static export)
- Styling: Tailwind CSS
- Hosting: Static deploy to CDN (Cloudflare Pages / Vercel)

## Milestones
1. Design mockup and content outline
2. Landing page MVP
3. Blog + documentation
4. Contact form and analytics
5. Launch and monitoring

## Success Criteria
- Lighthouse score > 90 on all metrics
- < 100ms TTFB
- Zero dependencies with known vulnerabilities
